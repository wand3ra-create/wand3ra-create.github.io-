Module 01

What Even Is a Robot?

Before you touch a single wire, you need to understand what you're actually building.

By the end of this module

  • Define what a robot actually is in simple terms
  • Identify the three things every robot does
  • Recognize robots in everyday objects around you

1.1Forget What Movies Told You

When I talk sbout robots, you probably picture something from a movie with metal legs, glowing eyes, maybe trying to take over the world.And that's probably what brought you here

But that is only a specific type of robot (And you will be able to hsve your own in due time. But for now :

A robot is anything that senses its environment, makes a decision based on what it sensed, and then acts.

1.2The Three Things Every Robot Does

Sense — A robot needs to know something about the world around it. It does this through sensors. A sensor converts something physical into an electrical signal.

Decide — Once the robot has information, it processes it. This is usually a small chip called a microcontroller. It runs code that says "if I sensed this, then do that."

Act — After deciding, the robot does something. The components that carry out actions are called actuators — motors, speakers, LEDs.

1.3You Already Own a Robot

Think about a phone you've used. Your phone is a robot. Not an exciting one, but technically — yes.

  • It senses — the accelerometer detects when you tilt it.
  • It decides — the processor runs code that says "screen tilted sideways, switch to landscape mode"
  • It acts — the screen rotates.

Try This — No Components Needed

Find three things around you that fit the Sense → Decide → Act pattern. A mousetrap counts. A door on a spring counts. If you can do this, you already think like an engineer.
